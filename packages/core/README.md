# @membrana/core

Ядро монорепо Membrana. Содержит базовые типы, контракты и утилиты, на которых строятся все остальные пакеты.

## Что делает пакет

- Определяет **базовые типы**: идентификаторы, временные метки, результаты операций.
- Описывает **доменные контракты**: интерфейсы сущностей, общих для расписания и устройств.
- Предоставляет **утилиты**: логгер, валидаторы, обработка ошибок.
- Фиксирует **иерархию ошибок**: `DomainError`, `ValidationError`, `NotFoundError`.

## Зависимости

**Никаких внутренних зависимостей.** Этот пакет — корень графа.

## Установка (внутри монорепо)

```json
{
  "dependencies": {
    "@membrana/core": "*"
  }
}
```

## Использование

```ts
import { type Id, type Result, ok, err, DomainError } from '@membrana/core';

function findUser(id: Id): Result<User, DomainError> {
  // ...
  return ok(user);
}
```

## Публичное API

Всё, что доступно потребителям, экспортируется из `src/index.ts`. Подробности — в JSDoc внутри исходников.

| Категория | Экспорты |
|-----------|----------|
| Типы | `Id`, `Timestamp`, `Result<T, E>` |
| Утилиты | `ok`, `err`, `isOk`, `isErr`, `logger` |
| Ошибки | `DomainError`, `ValidationError`, `NotFoundError` |

## Принципы

1. Pure functions only.
2. Никакого глобального состояния.
3. Стабильный API — breaking changes отдельным major-релизом.
