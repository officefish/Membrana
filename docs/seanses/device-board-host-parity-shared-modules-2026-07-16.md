# Консилиум: единый контур регистрации plugin-узлов device-board для client / cabinet / electron-студии

**Повестка:** Плагины борда лежат в `apps/client/src/plugins/*` и регистрируются только из `apps/client/src/main.tsx` через `registerClientModules()`. Кабинет (`apps/cabinet`) регистрацию не вызывает — палитра борда там сведена к базовым узлам `@membrana/device-board`. Нужно вынести регистрацию узлов/плагинов в общий контур так, чтобы борд в кабинете и в electron-студии имел ту же палитру, что песочница client, не нарушив граф зависимостей `ARCHITECTURE.md` и разные board-state контроллеры хостов.
