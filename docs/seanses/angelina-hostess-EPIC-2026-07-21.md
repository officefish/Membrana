## Вердикты (ратифицированы)

### M1 · C — каноны и скиллы
Создать **`membrana-morning-ritual`** (отдельный скилл утра: пред-ритуальный порядок,
цепочка с Ангелиной, два гейта, запрет магистрали от скрипта; триггеры «утро»/«ritual:day»);
из developer-rhythm утро **вычеркнуть** (ссылка, не копия; один атомарный PR). Статусная
механика: `status: live/deprecated/superseded` + `supersededBy` (DAG, live-терминал);
superseded не читается — переход по ссылке. **Failover-стоп** (недоступен новый → СТОП,
старый не замещает). `docs:verify-canon` — красный CI. Три предиката разбиения
(покрытие/непересечение/нет сирот). — `angelina-hostess-m1-canon-2026-07-21.md`

### M2 · B — сборка канона кодом
`frame()` собирает 5 секций кодом (вкл. слот «Список посылок»); LLM только per-slot
`fill → {text, status}`; **единый промпт-скелет запрещён**. `sign(doc, author)` +
`yarn canon:sign --author human` — ручная чеканка владельца подписывается штатно;
страж проверяет структуру/целостность, не авторство. Пустой слот → видимый «⚠», ретрай
одного слота. Гейт скелета остаётся страховкой (его красный = баг кода).
— `angelina-hostess-m2-assembly-2026-07-21.md`

### M3 · G — гейты в код
**Ружьё умерло как класс:** `send` — терминальный шаг, вызывается ⟺
`canSend = magistralChosen ∧ swallowApproved`; пути мимо предикатов нет.
`magistralChosen` — выбор владельца из замороженного снимка топ-3;
`swallowApproved` — `ownerAck === true`, черновик виден целиком до одобрения.
Идемпотентность: ключ = hash(payload, деньУтра). Предикаты — чистое ядро; `send` —
адаптер за границей. — `angelina-hostess-m3-gates-2026-07-21.md`

### M4 · H — хозяйка утра
Доступность = свойство активной сессии (`ritual:day` поднимает, `ritual:evening` гасит;
демон отвергнут). **Встреча — первая реплика дня** (имя, ревизия head, фронтир);
молчаливый старт запрещён. Абсорбция фронтира конкурента — **вызовом** гейтов-субагентов,
без копирования. Вычистка: `|entry| = 1`, атомарно, со следом.
— `angelina-hostess-m4-hostess-2026-07-21.md`

### M5 · GC — затирание следов (поправка владельца)
Каталог **`docs/void/`** — кладбище с эпитафиями (YAML-шапка: кто/за что/когда);
`gc.mjs` — детерминированный проход в вечернем ритуале после закрытия вердиктов;
**«затереть» = убить читаемость мёртвого пути, НЕ историю** (void растёт монотонно,
удаления запрещены). **Три барьера против возрождения** (шапка · индекс · grep-CI) +
`recent_void_penalty`. Граница: артефакты ≠ процессы (install-войны — контур
worktree-гигиены). GC шумный («перенесено 0» тоже печатается); **приговор — только
закрытый вердикт, GC сам не судит**. — `angelina-hostess-m5-gc-2026-07-21.md`

## Сквозное

Механика вместо дисциплины (что промпт не запрещает — агент сделает) · один источник
правды на каждый слой (скилл/канон/код) · честный след каждого решения (подписи, эпитафии,
провенанс) · история — актив.

## Реализация (за гейтом)

Порядок DAG: C (скиллы/каноны — docs, быстро) → B+G (код ядра утра, куски DoD #592) →
H (хозяйка) → GC (void + gc.mjs). Ночному/следующему агенту — по вердиктам, DoD в каждом
протоколе с владельцами-персонами.

## Процессные заметки заседания

Секретарский канал использован для M1/M2/M5 (комната системно роняет секцию посылок:
7 отказов за два дня; валидатор дважды ловил контрабанду — «вывод в посылках» — и
