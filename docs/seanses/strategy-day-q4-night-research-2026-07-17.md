[Музыкант]: Поддержу Дынина живой аналогией: это ровно декоррелированность детекторов из ансамбля 13.07. Yamnet ценен не точностью, а тем, что ошибается **не там же**, где DSP. [Верстальщик]: Тогда UI-контракт горизонта прямой: блок «Сигналы горизонта» — не стена, а список карточек, каждая с бейджем `external`/`insight`, датой, origin и одной строкой сути. [Структурщик]: Здесь жёсткий вопрос по границам. Ночной запрос к Perplexity — это внешний сетевой вызов, он **не живёт** в чистой стратегии. Иначе стратегия-функция станет нечистой и потянет ключи API в ядро. И второй запрет: `RawFinding` кладётся в артефакт `docs/…/research-night-<date>.md`, а стратегия читает markdown — не дёргает сеть сама.

[Математик]: Формализую перевод как отдельную чистую функцию, чтобы её тестировать без сети: `externalize(internalQuery) → externalQuery`. Вход — наш термин, выход — обезжаргоненный запрос. Тест-гард тривиален: словарь-стоп-лист внутренних аббревиатур (`ADR`, `UC`, `VDR`, `rt-N`), если хоть один просочился в `externalQuery` — тест падает. Если ночного артефакта нет (сеть упала) — утро деградирует явно, «сигналов сегодня нет», а не молча пустой блок.

[Teamlead]: Это ключевая граница, закрепляю её как инвариант. Глагол `highlight` из Q1: и ночной ресёрч, и форвард-инсайт входят в горизонт **только** как `signals[]` — read-only подсветка. [Математик]: И численный инвариант горизонта из Q2 держится: `signals[]` не влияет на `d/dt = 0`. Горизонт (gate, phase, criteria) стабилен по вехе; сигналы — это аннотация поверх, они меняются ежедневно, но **не двигают веху**. Формально: `horizon.criteria` — функция от `lastGateTransition`, `horizon.signals` — функция от `today`. Разные производные, не путать. Иначе ночной ресёрч начнёт незаметно перекраивать критерии — а это Q2-нарушение.

[Верстальщик]: Тогда на экране это два визуально разных яруса: сверху **веха** (стабильная, criteria — то, что не меняется день ко дню), ниже **сигналы** (свежие, датированные, схлопнутые по origin). Один главный акцент — веха; сигналы приглушены (`text-base-content/70`), это вторичный слой. Читатель сразу видит: веха — куда идём, сигналы — что нового услышали за ночь. Диагональ взгляда: сначала цель, потом шум-с-разбором.

Ночь пишет всё, что нашла (сырьё честное, с origin у каждого); утро при сборке `signals[]` схлопывает по origin-hash. ## Итоговое решение консилиума

| Вопрос | Решение |
|--------|---------|
| Куда ложится ночное сырьё и форвард-инсайты | В новое поле `horizon.signals[]` — **read-only подсветка**, отдельное от `criteria[]` и `task_ref[]` |
| Глагол | `highlight`, не `assign`: сигнал не становится задачей внутри стратегии |
| Переход `signal → task` | Только вне стратегии — человек или отдельный консилиум; стратегия не имеет права |
| Где живёт сеть | В `ResearchPort` (concrete-адаптер знает Perplexity); стратегия читает готовый markdown-артефакт `research-night-<date>.md` |
| Перевод жаргона | Чистая функция `externalize(internalQuery) → externalQuery` + стоп-лист внутренних аббревиатур (`ADR`/`UC`/`VDR`/`rt-N`); тест-гард ловит просочившийся жаргон |
| Когда порождается | **Ночью** (асинхронно, вне утреннего критического пути); утро только читает |
| Эхо-камера | Дедуп по `origin-hash` — на **утреннем чтении** (представление), не на ночной записи; ночь пишет полное свидетельство |
| Цена ресёрча | «Три» — верхняя граница, не догма: `min(3, budget)`, недобор помечается честно; счётчик прогонов + лог |
| Инвариант Q2 | `signals[]` не двигают веху: `criteria = f(lastGateTransition)`, `signals = f(today)` — разные производные, `d/dt(веха) = 0` держится |
| Деградация | Нет ночного артефакта / протух → честное «Ночной ресёрч не выполнен» с датой последней удачной ночи; не зелёная пустота |
| UI | Два яруса: веха (акцент) сверху, сигналы (приглушены, бейджи origin/дата/тип, схлопывание эхо) ниже; без чекбоксов |

**Definition of Done (если применимо):**
- `ResearchPort` + concrete Perplexity-адаптер; ночной запуск пишет `docs/…/research-night-<date>.md` с `RawFinding[]` (каждый с `origin`, датой, типом).
- Чистая функция `externalize()` + стоп-лист + unit-тест-гард (падает при просочившемся внутреннем жаргоне).
- Стратегия читает markdown + форвард-инсайты (`includeInsights=on`), собирает `horizon.signals[]` (read-only), дедуп по `origin-hash`, `min(3, budget)`-гард с логом.
- Инвариант: `signals[]` не мутируют `horizon.criteria`; тест на `d/dt(веха)=0` при смене сигналов.
- UI-блок горизонта: веха акцентом, сигналы вторым ярусом (бейджи origin/дата/тип, схлопывание по origin-hash, честная деградация, без чекбоксов) — по `DESIGN.md`.
- Все границы: ноль прямых импортов стратегии → Perplexity; сеть только через порт.
