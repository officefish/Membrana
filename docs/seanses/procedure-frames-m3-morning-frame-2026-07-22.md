[Структурщик]: Начну с термина, иначе фрейм поплывёт. «Подводка утра» — это омоним двух лемм, и их нельзя слить. Первая: **фрейм** — запись `{id, holder, pin?}` в `frames[]` манифеста процедуры `ritual-day` (вердикт M1). Вторая: **дверь** — конкретный отрезок в файле (`.claude/CLAUDE.md`, зеркала `developer-rhythm`, маршрут холодной сессии), на который фрейм ссылается через `pin` (вердикт M2). [Математик]: Формализую, иначе «состав отрезков» останется вкусовым. Пусть фрейм — множество пинов `F = {p₁, …, pₙ}`, каждый `pᵢ = {path, anchor{kind,ref}, segmentHash}` по вердикту M2. Инвариант целостности: `∀pᵢ auditPins(pᵢ).drift = false ⟺ фрейм здоров`. Аудит — чистая функция `auditPins(F) → Finding[]`, где `Finding = {pin, status ∈ {ok, drift, missing}, verb}`. Точка запуска — детерминированный проход, вход `F`, выход — список находок с глаголом ремонта; никакого `Date.now()`, дрейф считается сравнением `segmentHash` текущего отрезка с зафиксированным. Это тестируемо без DOM и аудиостека.

[Верстальщик]: С точки зрения потребителя фрейма — а потребитель здесь холодная сессия агента утром и человек в PR — фрейм обязан быть **виден как таблица дверей**, а не как россыпь пинов в JSON. Моя норма честности из `main-day-accuracy`: если дверь дрейфнула, отчёт аудита должен кричать об этом первой строкой — `status: drift` с бейджем `badge-warning`, `ok` — `badge-ghost`, `missing` — `badge-error`. Каждая находка несёт глагол ремонта видимым текстом, не только цветом. Дрейф без видимой эпитафии — мёртвая дверь.

[Музыкант]: По касательной, аудио тут не задет, но маршрут сигнала я узнаю — это как патч-лист сессии: список точек коммутации, каждая с проверкой контакта. Мой живой флаг из `angelina-hostess-m1-canon`: если дверь потеряна (`missing`), фрейм НЕ должен молча притворяться здоровым — это была бы мёртвая дверь, тихо заигравшая старую партию. Требую failover-терминал: `missing` в любом пине → аудит завершается статусом STOP с явной находкой, а не «в целом ок». Таймаут проверки ≠ здоровье пина.

Не переоткрываем контракт `frames[]` (M1 закрыт), не трогаем механику пина (M2 закрыт).
