## Итоговое решение консилиума

| Вопрос | Решение |
|--------|---------|
| Тип связи между тремя кандидатами | Направленная цепь (DAG без циклов): **1 → 2 → 3** |
| Ребро 1 → 2 | Кандидат-2 (зубы регламента) зависит от кандидата-1 (устройство персоны) **частично** — по публичному признаку «роль назначена на фазу», не по внутреннему устройству промпта/порога делегирования |
| Ребро 2 → 3 | Кандидат-3 (измерение) зависит от кандидата-2 **полностью**: домен измерения = множество следов, которые регламент обязал оставить в заданной форме |
| Обратные рёбра | Отсутствуют: 1 не выводится из формы своего следа; 2 — норма, а не функция метрик (иначе Гудхарт) |
| Фундамент | **Кандидат-1 «устройство ведущей персоны»** — без вердикта по нему регламенту некого обязывать подписью, измерению нечего атрибутировать |
| Независимость | Явно **нет**: ни одна пара не независима, все три связаны одной направленной цепью |
| Риск обратного ранга (3 раньше 2) | Метрики строятся над незафиксированной формой следа → меряют шум, ряд обнуляется при каждом рефакторе формы |
| Риск обратного ранга (2 раньше 1) | Регламент требует подпись несуществующей роли → «0 из 64 коммитов с аватаром» консервируется как норма, подписывать некому |

**Definition of Done (для вопроса O1):**
- Вердикт направленного графа зафиксирован как цепь 1 → 2 → 3 с обоснованием **каждого** ребра (сделано в теле).
- Фундамент назван явно (кандидат-1) с причиной, почему без него два других бессмысленны.
- Названы риски обоих обратных рангов (3↔2 и 2↔1) отдельными пунктами.
- Независимость рассмотрена и отвергнута явным вердиктом.
- Протокол не выносит суждений по существу самих кандидатов (только их порядок) и не касается запретных тем (удалённый контур персон, утренний ритуал, легаси-план дня, форма вердикта-токена).
