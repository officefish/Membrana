## Итоговое решение консилиума

| Вопрос | Решение |
|--------|---------|
| Список величин (формула + источник) | **`leadTime` = `closedAt − createdAt`** (карточка + артефакт); **`ownerlessRate`** = доля `leadPersona = ∅` (поле карточки); **`wip(p)`** = число open/in-progress карточек с `leadPersona=p` (срез реестра); **`throughput(p,w)`** = число закрытых карточек персоны за окно (артефакты); **`reworkRate`** = доля отозванных/переоткрытых закрытий (отзыв `acceptedBy`/регресс `headRev`); **`escalationRate`** = доля карточек со сменой `leadPersona` или переприсвоением `acceptedBy` (два снимка) |
| Загрузка персоны (численно) | **`wip(p)` = число висящих карточек по `leadPersona`.** Это загрузка **очереди намерений**, НЕ производительность. Честного источника производительности нет — заявлено явно |
| Парная метрика к каждой объёмной | `leadTime` ↔ `reworkRate`; `wip` ↔ `throughput`; `throughput` ↔ `reworkRate`. `ownerlessRate` и `escalationRate` — ограничители (норма 0% / вниз), пары не требуют |
| Намеренно неизмеримое | (1) индивидуальная производительность аватара (нет носителя, субагент обезличен); (2) вклад в LOC/коммитах (не носитель, инцидент 19.07); (3) «качество мышления» персоны (не редуцируется к следу) |
| Момент вычисления | Вечерний батч: два снимка реестра `[срез(t−1), срез(t))` + артефакты закрытия за окно → чистая функция → отчёт в репозитории. UI только читает |

**Definition of Done (C1):**
- Реализована чистая функция `computeTeamMetrics(prevSnapshot, currSnapshot, closureArtifacts, window) → TeamMetricsReport`; детерминирована (тот же вход → тот же выход бит-в-бит), без сети, без git-blame, без `Date.now()` внутри предикатов.
- Каждая объёмная величина в отчёте сопровождается своей парной метрикой; ни одна не выводится в одиночку.
- Загрузка персоны выводится как `wip(p)` с обязательной подписью «очередь намерений, не производительность».
- Отчёт несёт шапку: окно `[t−1, t)`, ревизии обоих срезов, множество учтённых артефактов закрытия.
- Секция «намеренно неизмеримое» присутствует в отчёте явным текстом (три пункта), а не умолчана.
