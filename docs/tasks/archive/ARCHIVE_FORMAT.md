# ARCHIVE_FORMAT — спека холодного контура `archive.jsonl`

> Р3 эпика [`registry-relocation`](../../meeting/registry-relocation/EPIC.md);
> вердикт [m3-cold-storage](../../seanses/registry-relocation-m3-cold-storage-2026-07-19.md).
> Реализация: `scripts/lib/cold-writer.mjs` / `scripts/lib/cold-reader.mjs`
> (блок `snapshot-cold-migration`, коворк `cowork-execution-registry`).
> Статус: **спека + офлайн-реализация на фикстурах.** Живой перенос 258 архивных
> записей — за гейтами владельца (дата рубежа законности, ратификация уточнения
> `insight-task-archive-storage`) и в спринт не входит.

## Назначение (три проверяемых пункта, M3)

1. **Независимость от поставщика** — подписка на внешний источник конечна во времени.
2. **Офлайн-детерминизм гейтов** — сеть в теле вердикта запрещена (M2); гейты
   читают локальную копию.
3. **Долговременность** — за горизонтом подписки.

Холод — не истина о статусе задачи (истина о движении — внешний источник);
холод — **архив условий**: что мы знали и решали на момент снимка.

## Форма

- Один файл `archive.jsonl`, **append-only**: писатель только дописывает строки,
  существующие байты неприкосновенны. Читается офлайн `.mjs`-скриптом без сети.
- **Единственный производитель — office-батч по триггеру.** Вечерняя архивация —
  сигнал триггера (`evening-signal`), своей дорожки записи не ведёт. Второго
  холодного источника не существует; запись с чужим `source` отвергается.
- Ступень 2 лестницы хранилищ (`insight-task-archive-storage`, 28.06; уточнение
  M3 ждёт ратификации владельца).

## Запись

Одна строка = одна запись = снимок карточки на момент архивации:

```jsonc
{
  "snapshot": {                        // honest-шапка — УСЛОВИЕ ПРИЁМКИ
    "capturedAt": "2026-07-19T18:00:00.000Z",  // дата снимка (ISO, UTC)
    "headRevision": "<git SHA head на момент снимка>",
    "source": "office-batch/trigger=evening-signal"  // всегда office-batch[/…]
  },
  "card": {                            // содержание карточки целиком
    "id": "issue-178-async-v2-reconciliation",     // адрес записи, обязателен
    "title": "…", "promptPath": "…", "githubIssue": 178, "linearId": "DRU-93",
    "status": "archived", "archivedAt": "2026-06-29", "archiveNotes": "…"
  },
  "domain": {                          // доменные поля — их нет во внешнем источнике
    "dependsOn": ["…"],                // граф зависимостей — ОБЯЗАТЕЛЕН (наш актив)
    "strategicWave": "W1",
    "leadPersona": "vesnin"
  }
}
```

### Правила приёмки (проверяет writer, покрыто тестами)

| # | Правило | Отказ |
|---|---|---|
| 1 | Honest-шапка `snapshot` с непустыми `capturedAt` (ISO) + `headRevision` + `source` | **запись без шапки холод не принимает** (M3, восстановлено финальным аудитом) |
| 2 | `source` = `office-batch` или `office-batch/…` | производитель один; вечер — сигнал, не писатель |
| 3 | `card.id` непуст | запись неадресуема |
| 4 | `domain.dependsOn` — массив | граф зависимостей обязателен |
| 5 | Полей движения (`state`, `stateType`, `transitions`, `history`) в `card` НЕТ | движение живёт во внешнем аудите, дублирование = синоним-слой |

## Критерии перехода ступеней (фальсифицируемые, M3)

| Переход | Предикат |
|---|---|
| 2 → 3 (генерируемый индекс) | `scan_time(archive.jsonl) > T_gate` (бюджет задержки гейта); индекс генерируется ИЗ jsonl, источник не заменяет |
| 3 → Postgres | `database is locked` чаще раза в неделю **при доказанном пересечении писателей во времени**, ИЛИ объём в единицах ГБ, ИЛИ второй **пересекающийся во времени** писатель |

Postgres и MongoDB на текущей фактуре (647 КБ, ε-конкуренция) — отклонены как
избыточные (вердикт M3, закрыт).

## Связь с миграцией (Р4)

Предикат `inCold(r)` критерия `wellArchived(r) = inCold ∧ hasPassport ∧
archivedAt ≠ null` читается отсюда офлайн (`cold-reader.mjs: coldIds`).
Классы долга (`debtClass`) вычисляются ДО переноса — см. `scripts/lib/debt-classify.mjs`.
