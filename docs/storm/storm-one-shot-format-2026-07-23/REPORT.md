## Что родилось

Формат **«one shot»** — седьмой в ряду [`FORMATS.md`](../../FORMATS.md), различающийся с
шестью существующими не числом команд и не динамикой, а **числом проходов**: один заход
без итераций. Оформляется как **процедура во фреймах**, а не как регламент-монолит.

### Первый фрейм — целиком

```
находка (агент сам / прецедент)
   → запрос штампа S у тимлида
   → чистая функция по критериям подтверждает «да, это S»
   → инлайн чек-лист плана
   → согласование и ратификация владельцем
   → исполнение до «весь код дописан»
```

### Подтверждённые тезисы

| # | Тезис |
|---|-------|
| **T1** | Предмет — новый формат разработки «one shot» |
| **T2** | Вход — маленькая проблема, найденная агентом **или** всплывшая из прецедента; работа **подобранная**, не заказанная |
| **T3** | Гейт допуска двойной, оба условия обязательны: размер **S** И не затрагивает работ на сервере |
| **T4** | Штамп S ставит **тимлид по запросу** — самоаттестация исключена |
| **T5** | Под тимлидом — **чистая функция по критериям**; суждение опирается на детерминированный предикат |
| **T6** | После штампа агент представляет план решения **инлайн чек-листом** |
| **T7** | Всё перечисленное — **первый фрейм** процедуры; формат строится фреймами |
| **T8** | Промах → **прецедент отказа** + альтернативы: шторм · инсайт · консилиум · спринт |
| **T9** | Успех → **согласие с обоснованием**; согласие без обоснования следом не считается |
| **T10** | Инлайн чек-лист идёт **владельцу** на согласование, владелец же **ратифицирует** |
| **T11** | **Ревью и мердж в формат не входят**; конец — когда весь код дописан |

### Три несущих свойства

1. **Формат не безгейтовый.** Агент подбирает работу сам, но допуск даёт тимлид (T4),
   а право исполнять план — владелец (T10). «Один выстрел» — про отсутствие итераций,
   не про отсутствие разрешения.
2. **Оба исхода оставляют след** (T8, T9). Отказ — прецедент с четырьмя маршрутами,
   успех — согласие с обоснованием. Контур замкнут: прецедент отказа становится входом
   T2 для следующего захода.
3. **Границы узкие с обеих сторон.** Вход отсечён гейтом S + не-сервер (T3), выход
   отсечён по «код дописан» (T11) — транспорт готового живёт снаружи и своими правилами.

## Сцепка с существующим (фактура репозитория, не тезисы)

| Элемент формата | Что уже есть |

| Чистая функция (T5) | идиом `planPrShip` / `checkBaseFreshness` / `classifyHttpProbe`: чистая логика отделена от исполнения и покрыта тестом |
| Размер S (T3) | поле `size` карточки реестра (`XS/S/M/L`) |
| Прецедент отказа / согласия (T8, T9) | контейнер `docs/precedents/` с мастерской (audit + decompose) |
| Вход «агент нашёл сам» (T2) | скилл `membrana-tooling-needs` — трение с эпизодом-evidence |
| Фреймы (T7) | спринт `procedure-frames` (#900): `frames[]`, пин отрезка (SHA + `segmentHash` + якорь) |
| Четыре маршрута отказа (T8) | ровно строки `FORMATS.md` — формат работает **маршрутизатором в форматы** |

## Осталось нерешённым

| Вопрос | Почему важен |
|---|---|
| ~~Что формат **отменяет**~~ — **отвечено хвостом Х1** ([`JOURNAL.md`](./JOURNAL.md)): отменяется **только регистрация** карточки в реестре | ответ пришёл после потолка, поэтому живёт хвостом, а не тезисом |
| **Сколько всего фреймов** в процедуре и чем кончается каждый | назван первый; граница второго (начинается на чек-листе или после исполнения) не проведена |
| **Критерии чистой функции**: только объём и пути — или в них входит «что правка собой держит» | правка размером S может держать гейт/промпт/регламент; тогда цена ≠ объём |
| **XS** попадает под one shot или у него свой, ещё более короткий путь | шкала уже существует, решение не принято |
| Форма **штампа тимлида** — реплика или артефакт со следом | штамп придётся предъявлять при разборе |
| Пересечение с **`procedure-frames` (#900)** | чужой ACTIVE-скоуп; назвать явно, не проглотить |

## Возражения питомца (голос, не основание гейта)

1. **Правка размером S не значит последствие размером S.** `.cursor/skills/` и
   `docs/prompts/` — не сервер и по объёму S, но промпт мастера управляет тем, как агент
   строит всё остальное; отставание такого промпта на такт стоило спринта (#993).
2. **Обоснование успеха ценнее прецедента отказа.** Отказ и так болит и запомнится;
   немотивированное «да» через месяц неотличимо от «да» по недосмотру — и порог тихо
   сползает вниз.
3. **Тезисы не проверялись на слом.** Одиннадцать тезисов за семь вдохов сложились
   складно, потому что диктовались, а не потому, что выдержали удар.
