# Сводный отчёт шторма: надёжность консилиума — «зелёный прогон при пустом содержании»

> Дом: `docs/storm/storm-consilium-reliability-2026-07-24/`. Открыт и закрыт 2026-07-24.
> Формат — дивергентный (беседа → тезисы), вердикта не несёт. Секретарь: Ангелина (инлайн).
> Голоса: 5 участников (Dynin · Kuryokhin · Ozhegov · Vesnin · Rodchenko) · питомец Фаррелл.
> Закрытие: ПОЛ («пора» владельца), вдох 3/7 (потолок не жгли). Живой конспект: [`THESES.md`](./THESES.md).

## Повод

Два прогона M0 заседания `membrana-leveling` (через openrouter/sonnet-4.6, Anthropic лимит
до 01.08) прошли все формальные признаки успеха — 20 реплик, вердикт, протокол, rt-6 «зелёный» —
но **содержание пустое**: комната дважды выдумала кандидаты повестки, соврала «повестка обрезана».
Провода целы (агенда 5 КБ < лимита 12 КБ, инжектится полностью), промпт раздут до 95 КБ.
Класс «объявленное ≠ работающее».

| **T4** ХРЕБЕТ | «Фрейм проводов консилиума» = **контейнер, не пункт списка**. T1/T2 и бывшие C1–C3 — три провода одного фрейма (канал · вход-сигнал · контракт вход↔вердикт). | синтез (C7) |

**Содержимое фрейма T4 (три провода):**

| Провод | Что несёт |
|--------|-----------|
| **Канал** (LLM-provider) | консилиум = LLM-процедура с пиновой цепочкой под класс вопроса; закрытый сильный канал → **видимая деградация в витрину**, не молчком (T1+T2+C3) |
| **Вход-сигнал** | под класс вопроса объявлено, что подводится: структурный → агенда, БЕЗ repo-контекста/RAG/памяти (C1) |
| **Контракт вход↔вердикт** | ID повестки — провод между входом и выходом; без ID страж rt-6 слеп, «зелёный» нефальсифицируем (C2) |

## Осталось нерешённым (в JOURNAL)

- Конкретная форма контракта канала (класс вопроса → допустимая модель) — вопрос заседания, не шторма.
- Соотношение с уже открытым заседанием `membrana-leveling` (то — про процедуру выравнивания;
  это — про надёжность самих LLM-процедур; смежны, но не одно).
- Диагностический специмен (два контаминированных M0) — кандидат в прецедент класса `tooling-gap`.

## Ответ Фарреллу (след согласия, S-S4)

Опровержения нет — питомец прав. У консилиума не было фрейма проводов; заводим его по #1094,
и пять разрозненных претензий становятся тремя проводами одного фрейма. Уступка, ставшая хребтом.
