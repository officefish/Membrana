## Что это за паттерн (одной фразой, языком доклада)

**Дом хранит — мастерская работает.** Контейнер (дом) — это пространство группы;
мастерская — поставляемая к нему оснастка, которой над домом работают. Мастерская
работает **минимум тремя глаголами** — *осмотреть* (audit), *разложить* (decompose),
*рассмотреть отдельный элемент*; сверх минимума растут специализированные доменные
инструменты. Инструменты мастерская **заказывает у поставщика — контейнера скриптов**,
который раздаёт наборы (киты). У процедурного дома третий глагол **полиморфен и
рекурсивен**: элемент сам контейнер, спуск идёт по вложенным домам до **якорного
документа**, ниже которого листа нет.

## Родовой узор (сведён из тезисов)

```
Контейнер скриптов ──раздаёт набор (кит)──▶ Мастерская ──работает над──▶ Дом (контейнер)
   [ПОСТАВЩИК]                              [ЗАКАЗЧИК/ПОТРЕБИТЕЛЬ]        [группа элементов]
                                                 │
                                    минимум 3 глагола: осмотр · декомпозиция · рассмотреть-элемент
                                                 │  (+ специализированные сверх минимума)
                                                 ▼
                        плоский дом → элемент = лист (одна ветка, одна карточка)
                        процедурный дом (2D) → элемент = под-контейнер; рассмотреть/разложить
                          рекурсивно спускаются до ЯКОРНОГО документа процедуры
```
