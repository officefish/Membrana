## Хвосты и связи

- **Терминологический сдвиг по ходу (Ожегов дважды):** «генератор»→«шаблон» (данные) + новый «генератор» (скрипт); и «валидность» (свойство) vs «валидация» (акт). Точность имён — несущая.
- **Гранула ≠ кристалл правды** — важное разведение: контейнер документов НЕ есть граф правды. Гранула = инструкция-несущая, кристалл = проверяемый факт.
- **office как control-plane** — каналы LLM (утренний провод) И валидация доков идут через office. Усиливает сегодняшний утренний диагноз (провод ритуала мимо панели) и долг попугая про мультипровайдера.
- **T21 (борд сети)** вырос из укуса Фаррелла про office-таймаут — наблюдаемость несущая для server-first, на который опирается вся валидация.
- **«Лиса в курятнике»** — антипаттерн родился из повторного лая Фаррелла (самовалидация); поднят владельцем в бестиарий. Это второй за день улов «pet → канон» (первый — фрейм временных файлов в шторме membrana-leveling).
