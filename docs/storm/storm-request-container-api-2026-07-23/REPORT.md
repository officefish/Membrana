## Подтверждённые тезисы

| # | Тезис |
|---|-------|
| Т1 | Пара «модель+провайдер» **выясняется у сервера** через мастерскую контейнера запросов. Процедура провайдера не выбирает и в себе не носит. |
| Т2 | **В процедурах агент остаётся без ключей.** Это отъём секретов, а не удобство маршрутизации. |
| Т3 | Вызов внешнего API — **не действие агента, а заявка**: агент формулирует запрос, исполняет сервер. «Процедура вызывает API» — неточно; процедура **просит исполнить**. |
| Т4 | Ответственность **не отменяется, а переезжает на сервер**: `countersign` остаётся нужен, билеты раздаются на сервере, у агента там **свой кабинет**. |
| Т5 | **Кабинет — зона полномочий** (не место хранения и не личность): агент делает ровно то, что в кабинете разрешено. |
| Т6 | **Серверную механику держит Фаррелл**: работа механическая, доверена роботу. |
| Т7 | **Контейнер запросов регулирует очередь** (порядок, задержка, отказ) **и считает токен-экономику**; Фаррелл работает глаголами его мастерской. |

## Куда едет

**Развилка: в заседание.** Конспект — вход заседания; дивергенция сходится к вердикту.

Вопросы, которые шторм породил, но решать не вправе (кандидаты в повестку комнат):

1. **Контракт кабинета** как зоны полномочий: что именно перечисляется, кто выдаёт, чем проверяется граница.
2. **Границы мастерской контейнера запросов**: какие глаголы, что делает Фаррелл её глаголами, а что — вне.
3. **Где на сервере живёт `countersign`** и кто там второй актор, раз ключей у клиента нет.
4. **Токен-экономика**: что считается единицей, кто читает счёт, чем он гейтит.
5. **Счёт вдохов на сервере** (хвост после потолка, см. ниже).

## Осталось нерешённым

- Т5 (кабинет = полномочия, не личность) **не закрывает** дыру актора: полномочия не дают отличимости субъекта. Второй актор на сервере по-прежнему не назван.
- Контейнер запросов **уже существует** (`docs/audit/llm-calls`, семья `audit-family` в атласе), но оснащён на два глагола из трёх — `inspectElement` отсутствует.
- Хвост после потолка: **вдохи тоже будет считать Фаррелл на сервере** — сказано владельцем, когда запись уже закрылась; тезисом не стало, уезжает входом в заседание.

## Честная пометка о ведении

Секретарь мерил вдохи **батчами** (Т1+Т2 и Т3+Т4 одним шагом) и насчитал четыре. Пересчёт механикой по событию дал **семь** — потолок был сожжён раньше, чем ведущая это заметила. Подгонка вскрыта самим механизмом; счёт в отчёте — машинный, не ручной. Это второй рецидив класса «вдох считает рука» после первого шторма.
