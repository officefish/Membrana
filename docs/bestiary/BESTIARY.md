> дифф с бестиарием и при находке **блокирует** с именем зверя.

## Как пополнять

Зверь заводится только с живым вещдоком (ссылка на прецедент/PR/протокол).
Пополнение — обычным PR; удаление зверей запрещено (append-only), устаревший
зверь помечается `⚰` с причиной. Недельный аудит-охотник — следующий шаг
инсайта, не этот документ.
