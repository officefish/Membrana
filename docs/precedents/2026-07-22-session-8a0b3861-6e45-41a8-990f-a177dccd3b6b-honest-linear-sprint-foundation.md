## Часть 4. Уроки / нормы (образец на будущее)

- **Туннель Linear:** локальный `LINEAR_API_KEY` + `HTTPS_PROXY` + `undici ProxyAgent` → прямой GraphQL из РФ без 403. Office в GraphQL не ходит (K1) — только локальная сессия/media-NL.
- **Норма зеркала (#984):** контейнер — руками (Linear-only); под-задачи НЕ создавать руками — GitHub-issue → зеркало → `task:register --issue N --linear DRU-M`. Ручные билеты = дубли.
- **Снос — архивацией, не hard-delete:** обратимо, освобождает квоту; фильтр по дате-расходности, а не «всё подряд».
- **Стена — доклад, не обход:** free-лимит назван вслух с развилкой, а не «тихо положил в другую команду».
- **Основание = три ноги:** движение (Linear-контейнер) + ответственность (GitHub-issue) + след (`leadPersona` реестра). План висит на карточках, не наоборот.
- **Промах чинить обратимо и называть:** каждый (дубли, junction, `.env`) — вслух и с возвратом, а не замолчать.
