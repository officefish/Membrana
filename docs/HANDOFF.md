## Как идёт выстрел (контракт первого фрейма)

```
находка → запрос штампа S у тимлида → предикат подтверждает S
   → инлайн чек-лист → ратификация владельца → исполнение до «весь код дописан»
```

- Вход — двойной гейт: размер **S** И не сервер. Пороги предиката: ≤ 8 файлов, ≤ 200 строк,
  запретные префиксы (`packages/background-*`, `deploy/`, миграции).
- Цепочка: окно 7 суток, ≤ 3 смежных выстрела — контрмера против дробления.
- **Штамп ставит тимлид по запросу**, самоаттестация запрещена. **Чек-лист ратифицирует
  владелец** — без его «да» исполнение не начинается.
- **Отменяется ровно одно** — карточка в реестре задач. Ветка, коммит, ревью, мердж,
  Linear живут как обычно, но **снаружи** формата: он кончается на «код дописан».

## Границы и предупреждения

- **Linear-карточки руками не заводить.** Автосинхрон создаёт их при заведении
  GitHub-иссью; ручная — дубль (урок DRU-235; сегодня повторён снова, шесть дублей
  DRU-397…402 отменены).
- **`pr:ship --issue N` ставит `Closes` безусловно.** Для PR, не несущего артефакты DoD, —
  `--allow-mention`, иначе иссью закроется ложно (живой случай #1022 / PR #1024, прецедент
  `2026-07-23-oneshot-issue-closed-without-procedure.md`).
- **`.env` у каждого worktree свой.** Ключи расходятся; перед сетевой работой сверять
  отпечатки. Linear и Anthropic — только через прокси; встроенный `fetch` его игнорирует,
  нужен `undici` + `ProxyAgent`.
- **Anthropic исчерпан до 2026-08-01.** Цепочка процедур падает на openrouter, кредиты
  есть; `yarn llm:probe` показывает маршруты.
- Формат в `docs/FORMATS.md` **не объявлять** — сначала живые прогоны.
