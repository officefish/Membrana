# Схема журнала памяти ведущей персоны (Ангелина)

> Вердикт M1 (DoD-2): ровно **три состава-поля**, провенанс = **«день + источник»**.
> Носитель: [`JOURNAL.md`](./JOURNAL.md). Журнал ведущей — отдельная конструкция,
> НЕ копия журналов советующих (`docs/virtual-team/memory/*.md`): там позиции,
> голоса и «как рассудила жизнь» — для ведущей эти вопросы бессмысленны.
> Журнал ведущей — **лог незавершённого**, его потребитель — утренний диалог
> следующего дня; читается без гадания.

---

## Три поля — не больше и не меньше

### 1. Незакрытые хвосты

Открытые единицы работы дня со статусом.

Формат записи:

```markdown
- [день: 2026-07-19 · источник: диалог с владельцем] Формулировка хвоста — статус: открыт
```

Допустимые статусы: `открыт` · `ждёт владельца` · `делегирован (писарь)` ·
`делегирован (аналитик)`. Закрытый хвост из поля **удаляется** (переезжает в
итог дня, если значим) — поле держит только висящее.

### 2. Напоминания владельца

Явные просьбы «не забудь про X» — дословно, без пересказа.

```markdown
- [день: 2026-07-19 · источник: реплика владельца в шторме] «не забудь про Y»
```

Напоминание живёт, пока владелец не снял его или оно не исполнено (тогда —
строка в итоге дня).

### 3. Итог дня

**Одна строка на день.** Чем кончился день — не отчёт, не список.

```markdown
- [день: 2026-07-19 · источник: закрытие сессии] Одной строкой: что состоялось.
```

---

## Провенанс

Каждая запись любого поля несёт ровно два атрибута:

| Атрибут | Форма | Правило |
|---|---|---|
| **день** | `YYYY-MM-DD` | календарный день сессии владельца; без времени суток; журнал не источник таймлайна |
| **источник** | свободная строка | откуда запись: реплика владельца в диалоге, возврат субагента (род, не имя — имени нет), процедура (шторм и т.п.) |

Запись без провенанса — брак и в журнал не вносится.

## Чего в журнале НЕТ (запреты схемы)

- позиций, голосов, оценок, «как рассудила жизнь» — формат советующих;
- ссылок на сеансы-обсуждения как провенанса (провенанс ведущей — день + источник, не консилиум);
- имён субагентов — их не существует; источник называет **род** («возврат аналитика»);
- четвёртого поля. Понадобилось четвёртое — это пересмотр вердикта M1, не правка схемы.
