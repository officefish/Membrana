## Дельта Phase 2

- **Подтверждено кодом (N1):** `EnsembleAnalysisRef` на вход `make-report-from-analysis`
  невалиден (`isValidSocketConnection`: union только на `DetectionAnalysisRef`).
  `MakeCombinedReport` — единственный зарегистрированный отчётный узел для
  нейро-анализа. **Нового узла НЕ потребовалось → BLOCKER.md не заводится.**
- **Подтверждено кодом (N2):** `validate-block-links` не требует подключения
  всех data-пинов → `analysis-2` молчит легально (прецедент Beta B6).
- **Уточнение к шву №2:** `branchCount` моей карточки = **1**, не 2 (у combined
  — 2: main + alarm). Заготовка сейчас несёт `0`.
- **Уточнение к шву №1:** мой `REPLACED_MVP_MAIN_NODE_IDS` (11 id) — надмножество
  Beta-шного (4 id). Если контракт унифицирует «список заменяемых узлов» как общий
  const — он обязан быть **per-block**, а не один на всех: сносимое у блоков
  разное by design.
- **Новых L-записей нет:** живой Run в Phase 2 не гонялся (гейт блока —
  структурный: тест + scoped CI). Живой Run = проверка **сборки**, планируется
  координатором на интеграции.
