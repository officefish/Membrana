# Стаб: фрагмент артефакта закрытия с полем приёмки

> **Гипотеза формы, не контракт.** Иллюстративный стаб блока `lead-persona`:
> показывает, куда, по ожиданию этого блока, ложится имя персоны, принявшей
> выход. Реальная форма артефакта закрытия — зона блока `units-trace-measure`;
> сверка — на Interface Consilium (Phase 3). В интеграционную ветку не мёржится.

```markdown
## Приёмка

| Поле | Значение |
|---|---|
| acceptedBy | `angelina` |
| головная ревизия | `<sha головы на момент LGTM>` |
```

Инварианты, которые блок `lead-persona` вкладывает в эту форму:

- `acceptedBy` — канонический идентификатор **персоны, принявшей выход**
  (нижний регистр, латиница — как у пятерых советующих);
- приёмка ставится только ПОСЛЕ `isValid = true` возврата, из которого собран
  выход; невалидный возврат следа не порождает;
- имени субагента-автора в артефакте нет и быть не может — субагент безымянен
  по контракту M1; требовать подпись = сочинять имя.
