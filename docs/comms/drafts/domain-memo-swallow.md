🌐 **Где живёт Membrana**

Навели порядок в «адресах» — теперь два домена, у каждого своя роль:

**membrana.space — продукт** (то, что для пользователей):
• **Главная** — скоро: описание, регистрация, скачивание клиентов
• **Документация** — `membrana.space/scenarios/docs`
• **Скачать десктоп** — `membrana.space/downloads`
• **Кабинет** — `cabinet.membrana.space` (уже работает)
• В планах — витрина пользовательских сценариев на `membrana.space/scenarios`

**mmbrn.tech — рабочая инфраструктура команды** (внутренняя кухня):
• `office.mmbrn.tech` — фоновые сервисы
• `panel.mmbrn.tech` — панель управления

*По-простому:* всё, что видит пользователь — под **membrana.space**; рабочие инструменты команды — на **mmbrn.tech**. Чётко, ничего не путается.
