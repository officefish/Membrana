# Шапка-пояснение для telegram-дайджестов

> Источник свёрнутой шапки (`blockquote expandable`), которую бот вшивает в каждый
> дайджест (#434). Редактируется без кода: скрипт ритуала читает секцию ниже как есть.
> Держать компактной — она входит в лимит сообщения Telegram (4096 символов).
> Полная версия — [`ALLY_PRIMER.md`](./ALLY_PRIMER.md).

<!-- digest-header:start -->
**Что такое Membrana:** сеть недорогих «ушей» — микрофон плюс мини-компьютер, — которые в реальном времени распознают малые дроны по звуку, без радара. Сам узел прост, вся сложность живёт на уровне сети; система наблюдает и предупреждает, ничем не управляет.

Проект ведёт владелец с виртуальной командой из пяти ролей (Teamlead, Структурщик, Математик, Музыкант, Верстальщик). Бот дважды в день присылает выжимки: утром — план, вечером — итоги с оценкой полезности дня. Полная памятка: [ALLY_PRIMER](https://github.com/officefish/Membrana/blob/main/docs/comms/ALLY_PRIMER.md).

*Мини-словарь:* **PR/merge** — порция изменений и её принятие; **Issue #N** — карточка задачи; **консилиум** — совет пяти ролей до кода; **детектор** — распознаватель звука дрона; **fusion** — объединение мнений детекторов в один балл; **S3** — упаковка готовых сценариев в каталог; **VDR** — проверка качества на полевом оборудовании.
