## 1 · Магистраль (L)

**Поставить три базовые процедуры на фрейм-рельсы:** `ritual-day`, `day-sprint`, `meeting` — наполнить полосу **`frames`** (автоцепочка) в их манифестах, держатели из канона персон, зуб `validateProcedure` зелёный.

Контракт это уже позволяет (`procedure-frames` F1 / #927 + `ritual-day-frames`): легальны три полосы очереди — `preflight` (гейт до), **`frames` (автоцепочка)**, `post` (ручной хвост). Элемент кадра — `{id, holder}` + опциональный непустой `pins[]`; **гранула реализована пином**, отдельного поля `granules` в схеме нет.

Состояние на утро: у `ritual-day` заполнена только полоса-страж (`preflight: morning-wiring`, держатель Ожегов, три двери с хешами отрезков); **автоцепочки нет ни у одной из трёх** процедур.

**Критерий успеха к вечеру:** у трёх процедур `frames` не пуста; `procedures:workshop --inspect` рендерит **цепочку**, а не паспорт манифеста; запись сделана **под лицом Ожегова** (фреймы процедур — его зона) с приёмкой, а не правкой мимо держателя.
