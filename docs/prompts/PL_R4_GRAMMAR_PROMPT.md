## Acceptance criteria

> Источник: вердикты заседания procedural-layer (см. Контекст). Чеклист приёмки:

- [ ] Словари kind и персон в файле правил Р3 (одна точка истины); feature вне словаря
- [ ] Чистая resolveHolder(explicit, branchPersona) → persona|CONFLICT|MISSING; explicit = слово при шипе ∨ leadPersona карточки при slug=id; ≥4 юнит-тестов + конфликт + сирота
- [ ] Валидатор имени ветки [персона/]kind/slug + теги формата; вне словаря → MISSING с подсказкой
- [ ] Провод в шип-скилл: шапка «тип · держатель (+источник) · заморозка», исходы текстом; CONFLICT/MISSING = возврат на доработку (T8)
- [ ] Предикат night-заморозки: мердж из-под night-ветки блокирован до флага «ночь закончилась» (T7)
