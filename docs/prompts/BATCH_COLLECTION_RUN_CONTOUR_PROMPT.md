## Почему это НЕ узел, а эпик

Консилиум (СТ-3, единогласно) отделил этот пункт от узлов наглядности PC-1/PC-2.
Улика №3 коворка: кейс «прогон детекторов по импортированной библиотеке»
невыразим — в палитре нет ни типа данных коллекции, ни узла-источника. Но это не
«добавить узел»: коллекция **ломает саму модель исполнения**.

- Текущая модель: источник — живое устройство (`microphone`), поток → анализ →
  отчёт, один каденс (live).
- Коллекция: источник — сохранённый набор, появляется **ИТЕРАЦИЯ** по N сэмплам;
  это офлайн/batch-прогон, антипод live-каденсу mic-detection и alarm-loop.

Смешивать live и batch в одной палитре без явной границы = сценарии, которые
никто не поймёт. Это новые core-контракты: тип `SampleCollectionRef`,
узел-источник, узел-итератор (`for-each-sample`), входное устройство `collection`
(как `microphone`), и рантайм, умеющий гонять граф по набору вместо одного потока.

## Почему следующий цикл, а не сейчас

«Прогон по библиотеке» ценен не для FREE-витрины, а для **валидации детекции на
реальных данных** — это ровно пост-FREE полевая фаза (VDR hard-gate, полевые
записи). Тащить его до отгрузки FREE — против инварианта владельца (живой дрон /
полевая валидация — смысл отгрузки, а не гейт перед ней). Эпик **зарегистрирован,
не потерян**, но стартует после отгрузки FREE и по решению владельца.

## Контур работ (набросок — уточняется консилиумом при старте)

1. Консилиум-гейт: новая execution-модель live↔batch, границы, core-контракты.
2. Тип данных `SampleCollectionRef` + входное устройство `collection`.
3. Узел-источник коллекции + узел-итератор `for-each-sample` (семантика итерации).
4. Рантайм batch-прогона (граф по N элементам, агрегация результатов).
5. UI: где batch-сценарий, как отделён от live; связь с клиентским модулем
   «Библиотека сэмплов» (импорт/разметка — там, не в графе).

## Возможный мостик (не эпик)

`get-collection-sample` — один сэмпл из клиентской библиотеки как `AudioSampleRef`
в обычный live-граф. Маленький узел, БЕЗ итерации/batch. Вводить **только если
появится кейс**, которому он честно нужен (принцип «узел от кейса»). Не
превентивно, не часть этого эпика по умолчанию.
