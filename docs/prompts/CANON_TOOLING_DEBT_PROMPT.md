### ti-1 · ADR-гигиена

Дубль номера — симптом. Механизм ADR **вчера объявлен каноном** вместо мёртвой ветки
`vesnin` (#499), и выглядел заброшенным: реестр в `docs/adr/README.md` нёс **2 записи из 7**,
именование разъехалось (`0002-...` против канонического `ADR-NNNN-...`), два разных решения
носили номер 0005.

**Правило разведения:** 0005 остаётся за тем, на кого больше ссылок. `panel-users-store` —
код (`apps/panel/src/lib/roles.ts`), `BACKGROUND_OFFICE_DEPLOY.md`, эпик-промпт PU1 (он этот
номер и планировал). `night-narrative-provider-chain` → **ADR-0007** (одна ссылка).

### ti-2 · 26 живых фаз реестра

Точечная часть проблемы 189 фаз-носителей `githubIssue` эпика. Чиним **только активные**:
`githubIssue` → `null` (прецедент #468). Связь с эпиком держит `parentEpic`, она не трогается.

### ti-3 · `--report <файл>` для `repo:clean`

Отчёт печатался только в stdout; прогон через `| tail -18` потерял текст двух ошибок.

### ti-4 · `process.exit()` в горячих скриптах

**Сначала измерить, потом чинить.** Установлено замером: рвёт **не stdout вообще**, а выход
при **живых сетевых сокетах**. `--help`-пути не падают (3 прогона подряд, exit 0) — их не
трогать. Падают выходы после HTTP-вызова.

В `code-review.mjs` найден костыль `sleep(150)` перед выходом, стоявший **только на успешном
пути** — потому и падал ровно путь ошибки (`Anthropic без кредита` → 127 вместо 1).

### Тесты

| Область | Минимум |
|---------|---------|
| `--report` | путь берётся следующим аргументом; без флага `null`; `--report` без пути → undefined |
| `createReporter` | копит строки и печатает; **stderr тоже попадает в отчёт** (из-за его потери флаг и появился) |
| Реестр | живых фаз-носителей `githubIssue` эпика — ноль (регрессия по живому реестру) |
| ADR | все ссылки на `docs/adr/*` резолвятся |

### Definition of Done

- [ ] ADR: номера уникальны, реестр полон (все 6 + исторический), имена по канону, ссылки не битые.
- [ ] `yarn task:close-github:dry`: **живых** фаз-носителей ноль.
- [ ] `yarn repo:clean --report <файл>`: файл полон при обрезанной пайпом консоли; без флага поведение не изменилось.
- [ ] Горячие скрипты возвращают **1, а не 127** на пути ошибки — проверено **живым прогоном**, а не рассуждением.
- [ ] `yarn test:scripts` + `yarn docs:verify-canon` — зелёные.
