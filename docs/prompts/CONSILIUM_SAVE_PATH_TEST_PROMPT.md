## Контекст

Задача — из отказа one-shot 23.07. Первый прогон M0 заседания `tasks-workshop` отработал
28 реплик комнаты и умер при сохранении: `ReferenceError: model is not defined`
(`scripts/consilium.mjs`, вызов `wrapSeanseFile`). Регрессия перехода на procedure channels
(#1021): одиночной переменной `model` больше нет, звено выбирается на лету. Починено PR #1032
(`model: answeredBy ?? chainLabel`), **теста на путь сохранения нет**, а дефект пережил CI,
ревью и мердж, потому что ломался ПОСЛЕ генерации ответа — деньги потрачены, результат выброшен.

Формат one-shot на это отказал ПРАВИЛЬНО (прецедент
`docs/precedents/drafts/2026-07-23-oneshot-1051-refusal-save-path-test.md`): предложенный
S-тест закрывал класс функции (`wrapSeanseFile`), а не класс регрессии — точка падения жила
в `main()` и тесту не видна. Причина невидимости: в `scripts/consilium.mjs` **ноль
экспортов**, а `main()` зовётся безусловно — любой импорт прогоняет консилиум. Образец, как
надо: `scripts/meeting-audit.mjs` — там гард `if (process.argv[1]?.endsWith('meeting-audit.mjs')) main()`
прямо подписан «без него импорт из теста прогонял бы аудит».

**Не трогаем:** саму цепочку procedure channels, формат протокола и метаданных сеанса,
пометку канала (`#1047` уже её проверяет).

## Промпт целиком (для вставки агенту)

### Кто ты

Координатор виртуальной команды Membrana. Ведущий — **Dynin** (Математик, чистые функции и
их тесты), support — **Vesnin** (Teamlead, границы). Перед кодом — план (1–2 абзаца + файлы).

### Что построить

Сделать путь сохранения протокола консилиума **тестируемым** и покрыть его тестом с мокнутой
цепочкой, включая ту ветку, где сегодня жил `ReferenceError`.

### Definition of Done

- [ ] **Гард запуска** в `scripts/consilium.mjs`: `main()` вызывается только при прямом
      запуске (`process.argv[1]?.endsWith('consilium.mjs')`). Без него файл не импортировать.
- [ ] **`wrapSeanseFile` и вычисление метки модели вынесены** в чистую функцию
      (`seanseModelLabel({ answeredBy, chainLabel })` или эквивалент), покрываемую без сети.
- [ ] **Путь сохранения исполняется тестом с мокнутой цепочкой** — не только `wrapSeanseFile`:
  - [ ] ответило ПЕРВОЕ звено → метка = имя первого звена;
  - [ ] ответило ВТОРОЕ звено (fallback) → метка = имя второго;
  - [ ] не ответил никто → метка = `chainLabel`, без `ReferenceError`.
- [ ] **Обе ветки записи** покрыты: `docs/seanses/` (гейт чист) и `docs/seanses/rejected/`
      (гейт заседания упал) — запись не обращается к переменным вне области видимости.
- [ ] Регресс-защита: тест падал бы на коде ДО PR #1032 (доказать, что ловит этот класс).
- [ ] `node --test` зелёный; `yarn turbo run lint typecheck test --continue` в scope — зелёный.
- [ ] LGTM Teamlead.

### Out of scope

- Переписывание цепочки procedure channels.
- Изменение формата протокола / метаданных сеанса.
- Пометка канала (проверяется #1047).
