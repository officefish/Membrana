## Промпт целиком

Верстальщик (**Rodchenko**) — видимость для ревьюера/шип-гейта (текст, не UI apps):

1. Короткая линза в AGENT_PROMPT или `analysis/…-review-lens-….md`: как за 30с понять жанр ветки из ассортимента
2. Связь с шапкой шип-гейта (тип · держатель · заморозка) — **документарно**, без кода Р4
3. Провод в `membrana-code-review` / CONTRIBUTING «Гигиена веток» — одна ссылка

**DoD:** ревьюер может открыть контейнер и выбрать представителя жанра; нет дубля движка грамматики.
