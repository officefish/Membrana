## Что построить

Подключить сессии агентов (Claude Code) как **свидетельство** для ритуала: новый вход `~/.claude/projects/*.jsonl` в **существующий** `persona-memory-extract.mjs` — без второго экстрактора и без нового пакета.

**Класс — свидетельство, не источник истины** (термин Структурщика, снимает половину риска сразу). Иерархия 16.07 остаётся: **маркер в коде первичен**. Экстракт — третий класс рядом с «код» и «Issue»: **гипотеза-для-сверки**, вход для `main-day-probe`. Свидетельство **сверяется**, а не исполняется.

## Прецедент (живой — на нём и демонстрировать)

Сессия 13.07 знала, что yamnet слит в combined. Ритуал 16.07 не знал — и назначил магистралью переписать готовое ядро `fuseDetectorConfidences`. Issue-статусы врут при забытом `Closes`, снимок приоритетов замерзает; сессии знают, что **реально делалось**.

**Контрриск, который нельзя забыть:** сессия знает **намерение**, а не результат. «Я слил yamnet» ≠ «yamnet в main». Агент, уверенно рассказывающий о сделанном, — ровно тот источник, который подвёл 16.07. Поэтому только гипотеза, только со сверкой.

## Решённые риски (не переоткрывать)

| Риск | Решение |
| --- | --- |
| Приватность: ключи, `.env` в сыром вводе | Секрет-скан до дистилляции; фрагмент выбрасывается; в репо только дистиллят |
| Достоверность: намерение ≠ результат | Класс «свидетельство» → гипотеза для сверки, не факт для плана |
| Третье эхо: агент писал → агент дистиллировал → агент спланировал | `origin = sha(fragment)` в общем дедупе |
| Шум: 114 сессий, в основном тупики | Готовый отбор top-N по recency+importance, токен-бюджет |
| Локальность: `~/.claude` не версионируется | Сырьё локально — осознанно; в репо дистиллят |
| Разнородность: Claude Code / Cursor / Codex | Начинать с Claude Code (114 сессий уже есть), не проектировать под остальных вслепую |

## Нормы эпика (обязательны)

- **Живая демонстрация на прецеденте выше** — unit-тестов недостаточно. PG3 #533: гейт соврал при 12 зелёных тестах ядра, поймал только реальный прогон.
- **Регрессия проверяется возвратом бага** — верни поломку, тест обязан покраснеть.
- **Сначала проверь, что инструмента нет.** Здесь это особенно остро: `persona-memory-extract` уже делает всё, кроме чтения jsonl.
- Новых пакетов нет; `@membrana/core` не трогаем.
