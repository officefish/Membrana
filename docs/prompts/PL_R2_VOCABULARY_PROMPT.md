## Acceptance criteria

> Источник: вердикты заседания procedural-layer (см. Контекст). Чеклист приёмки:

- [ ] Единственный машиночитаемый источник словаря в доме слоя; 4 статьи по 5 полей (имя, определение-предложение, род, маркер @cat:, предикат)
- [ ] Роды по m2a-rod: задача=объект, ответственность=отношение, команда=ориентир, цель=ориентир; леммы родов записаны
- [ ] VOCABULARY.md генерится скриптом (пометка generated, ручная правка ловится)
- [ ] Чистый check(layerText, vocabulary) → violations[{term,location,reason}]; «0 нарушений» — явный пустой список; юнит-тесты на ложное срабатывание и пропуск
- [ ] checkGenus(op) поверх check(): effect ∈ allowed(род); немаркированные мутации ориентиров — честный хвост аудитора в отчёте
