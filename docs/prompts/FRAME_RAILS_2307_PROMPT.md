## Цель

Поставить три базовые процедуры — **`ritual-day`**, **`day-sprint`**, **`meeting`** — на фрейм-рельсы: наполнить полосу **`frames`** (автоцепочка) в их манифестах, держатели из канона персон, зуб `validateProcedure` зелёный.

**Критерий успеха:** у трёх процедур `frames` не пуста; `procedures:workshop --inspect` рендерит **цепочку**, а не паспорт манифеста; запись сделана под лицом держателя (фреймы процедур — зона Ожегова), а не правкой мимо него.

## Контракт (снят с кода, не с канона)

Легальны три полосы очереди: `preflight` (гейт до) · **`frames` (автоцепочка)** · `post` (ручной хвост). Элемент кадра — `{id, holder}` + **опциональный непустой** `pins[]`; пин — `{path, anchor{kind, ref}, segmentHash}`. Отдельного поля `granules` в схеме нет: **гранула реализована пином**, фрейм несёт стек пинов.

Состояние на утро 23.07: у `ritual-day` заполнена только полоса-страж (`preflight: morning-wiring`, держатель ozhegov, три двери); автоцепочки нет ни у одной из трёх процедур.
