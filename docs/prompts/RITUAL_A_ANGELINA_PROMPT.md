## Контекст

Ритуал (дневной/вечерний) — линейная цепочка скриптов без координатора: узлы не
перепроверяют свежесть и структуру, сбой уходит под зелёным кодом («молчун»). Заседание
`ritual-refactor` (M0..M5) запроектировало рефакторинг; корень DAG — **A: Ангелина-**
**координатор** (M1). Она — сквозной страж всего каскада документов (`STRATEGY_DAY`,
`DAILY_STANDUP`, `MAIN_DAY_ISSUE`, `DAILY_CODE_REVIEW`, `DAILY_AUDIT`, доклад, дайджест снов).

**Не трогаем:** генерацию самих документов (K/S/R/D — отдельные спринты), граф правды,
форму вердиктов заседания.

**Связанные:** [`ARCHITECTURE.md`](../ARCHITECTURE.md) (границы), эпик
[`ritual-refactor-EPIC-2026-07-20.md`](../seanses/ritual-refactor-EPIC-2026-07-20.md).

**Движок задач:** отразить спринт в Linear-сценарии — Linear **полностью доступен через
медиа-сервер**; исполнитель = делегат (`delegate`).

## Что построить

1. **`orchestrateCascade(graph, snapshot)`** — единственная публичная точка входа.
   Детерминированный топологический обход каскада (tie-break по `id`), для каждого узла:
   проверка провенанса + свежести входящих рёбер.
2. **Страж-функции** (внутренние, экспортируются для тестов): `freshness(producer, edgeRead)`
   → `fresh|stale|unknown`; `provenanceProblem(snap)`; `topoOrder(graph)`.
3. **Предъявление** узла: `presentNode(id, r)` — строка `состояние · автор · страж · digest`.

## Архитектура / контракт

| Слой | Путь | Ответственность |
|------|------|-----------------|
| Чистое ядро | `scripts/lib/angelina-cascade.mjs` | сравнение версий/digest, топосорт, провенанс — без сети/DOM/mtime |
| Тесты | `scripts/angelina-cascade.test.mjs` | все инварианты DoD, без моков |
| Адаптер (позже) | `scripts/*.mjs` | `version = git log -1`, `digest = orderedDigest(содержимое)` → снимок |

**Инварианты (M1):**
- Свежесть детерминирована: версия по `git log -1` (не `mtime`), digest = упорядоченный sha256.
- `unknown` («не проверено») **≠** `fresh` — явная нейтральная метка, не мёртвая дверь.
- Любой `stale` **или** проблема провенанса → каскад **не ok** (ненулевой exit у адаптера),
  громко; никаких `|| true`.
- Провенанс `{author, guard, digest, readAt}` обязателен; `author ∈ {5 персон, человек}`,
  субагент запрещён.
- Ангелина **не редактирует** документы и **не судит пользу** — только свежесть+структуру.

**Запрещено:** сеть/DOM/`mtime` в ядре; редактирование документов; провенанс задним числом.
