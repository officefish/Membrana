# Эпик: генератор стратегии дня — от списка задач к горизонту

**Статус:** спека собрана из пяти вердиктов консилиума и слова владельца 17.07. Кода нет.

## Зачем это, в одной мысли

Стратегия дня **должна быть первой** и существовать ради **одной** цели: **определить дальние границы дня — расширить горизонт задач**. Потом MAIN_DAY_ISSUE централизует основной поток, и стратегия даёт ему **цели для дополнительных задач, которые поднимаются над ежедневной рутиной и ведут проект к долгосрочным целям**. Стендап сцепляет всё окончательно, распределяя нагрузку по команде.

Сегодня она этого не делает. Расследование ([`strategy-day-investigation-2026-07-17.md`](../seanses/strategy-day-investigation-2026-07-17.md)): **из семи задуманных функций работает одна** — порядок запуска.

## Решения владельца 17.07 (закрывают дыры Q5)

**1. «Своевременность инсайта по логам» = ВЕХА БЛИЗКО **И** ОБЛАСТЬ МОЛЧИТ.**

Инсайт подсвечивается, если он привязан к ближайшей вехе **И** git молчит о его области N дней. Это буквально замысел владельца в двух конъюнктах: «**поднимаются над рутиной**» (молчит) **и** «**ведут к долгосрочным целям**» (веха).

> **Отвергнуто и почему:** «логи **трогали** его область» — буквальное чтение «соотносит с логами», но оно **инвертирует замысел**: подсветка пошла бы **за** тем, что уже делаем, усиливая рутину вместо подъёма над ней.

**Цена, названная при выборе:** инсайты без привязки к вехе **не всплывут никогда**, а из 27 накопленных привязаны единицы. Простановка вех — работа, и она в этом эпике.

**2. Дикого слота в ночи НЕТ — дикое приходит утром само.**

Ночь = **100% `derived`** (сны из пар кристаллов). Внешнее не выбирается нами: **входящее** (17.07 — Антон прислал датасет, Денис устройство; никто не спрашивал) относится к **«логичному настоящему» утра**, где сознание взвешивает его против сна. **80/20 из Q4b растворяется.**

**Цена, названная при выборе:** если неделю никто не пишет — внешнего нет вовсе, рынок мы не смотрим. Принято сознательно.

**3. Отсутствующий вход — ПОМЕЧАТЬ.** *(вывод агента, не команды — оспорить дёшево)*

Посылки: (а) владелец 17.07 — «протухшее не блокирует; гейт, который краснеет постоянно, отключают» → **не ронять**; (б) уцелевший пункт Q4 — «нет артефакта → честное „не выполнен“, не зелёная пустота» → **не молчать**. Остаётся **помечать**: артефакт обязан сказать «канал мёртв».

## Работы

**S1. Ядро горизонта.** `collectHorizonInputs(horizon, {insights, research}) → {highlights[], provenance[]}` в `scripts/lib/`. Чистая, детерминированная, тест без сети. Дедуп по `origin-hash` — **переиспользовать `originHash` из `lib/truth-graph.mjs`** (перенесён из `main-day-probe` #538), не писать заново.

**S2. Горизонт = веха.** Вырезать `HORIZON_LABEL`/`nextDay`. `{gate, phase, criteria[]}`, `since = max(lastGateTransition, now − N)`. Инвариант `d/dt = 0` — тестом.

**S3. Своевременность.** Предикат `isTimely(insight, horizon, gitLog)` = веха близко **И** область молчит N дней. Простановка вех у инсайтов (27 файлов) — отдельная карточка, ручная.

**S4. Каналы.** Флаги из недельной в параметры `collectHorizonInputs`. Один читатель. Отсутствующий вход → **пометка «канал мёртв»** в артефакте, не тишина и не падение.

**S5. Мёртвый крон.** Решить судьбу `weekly-analyzers-research.yml`: причина 9 падений неизвестна. Вердикт Q5 неявно отправляет его в отставку (ночной ресёрч занимает место), но крон **оставлен стоять** — назвать явно: чинить, удалить как фикцию или заморозить вместе с недельной.

**S6. Ночной контур.** `pickTopic` (100% `derived` — пары кристаллов) → `externalizeQuery` → Perplexity → артефакт с фронтматтером → `nightYield`. Слои **поверх** `night-hunt`, не переписывая его.

**S7. Стратегия читает граф правды.** Сейчас grep = 0. Кристаллы — источник пар для `pickTopic` (S6) и посылок горизонта.

## Definition of Done

- `collectHorizonInputs` — чистая, детерминированная, тест без сети.
- **Дедуп:** вход с тремя строками одного `origin-hash` → выход `n = 1` (**тест на эхо 06.07**).
- `externalizeQuery` ловит жаргон на фикстуре (`ADR`, имя пакета) → **явный FAIL с читаемым сообщением**; отклонённый вопрос — видимое состояние, не тихий сбой.
- Stale срабатывает при `age > веха` текущего горизонта; помеченный highlight рисуется приглушённо с бейджем даты.
- Флаги `includeInsights`/`includeAnalyzersResearch` **удалены из тела недельной**, стали параметрами; **один читатель**.
- Все входы вливаются как `highlight`; **ни один канал не имеет права на `assign`** (проверка границы в ревью).
- `isTimely` — тест на обе ветви: веха далеко → не подсвечен; область шумит → не подсвечен.
- **Отсутствующий вход даёт видимую пометку**, а не `null` и молчание — тест на регресс двух месяцев.
- Инвариант `d/dt = 0` — тестом: смена календарной даты **не меняет** горизонт.
- Пакетный граф `ARCHITECTURE.md` **не изменён** (только `scripts/lib/` + чистое ядро дедупа).

## Чего в эпике НЕТ

Формат `roadmap-state` · «совершенно иные процессы» вечера · **недельная как таковая** (заморожена словом владельца) · причина падения крона (**отдельная работа**, S5 решает только судьбу) · привязка процедур к графу правды (владелец отложил: «нужно хорошо обдумать») · фоновый агент-аудитор графа (владелец: регистрируем инсайтом **после** эпика).
