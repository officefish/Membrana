## Acceptance criteria

> Источник: вердикты заседания procedural-layer (см. Контекст). Чеклист приёмки:

- [ ] Единый файл правил: layer(path) + разрешённые межслойные паттерны; один файл — один SHA
- [ ] Чистая checkLayerDirection(importGraph) → violations[] на статическом графе импортов (ранги 0<1<2, ссылки только вниз); слепая зона динамических ссылок названа
- [ ] Блокирующий CI-гейт: |violations|>0 → exit 1; ошибка построения графа — отдельный exit-код
- [ ] Контракт кит-манифеста как обязательного входа ревью задокументирован (BLOCK без манифеста/с нерезолвящимися ссылками)
- [ ] Заведён docs/procedures/attribution/ (контейнер-определение парсера, manifest-only); формат вердикта с rulesSha описан в README
