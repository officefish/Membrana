`_ssh-office-config` (#349) и повторяет форму `_ssh-media-exec` — новой обвязки
ssh2 не заведено, слой один. Чистые функции (`stripCarriageReturns`,

покрыты тестами без сети. Гард `process.argv[1]?.endsWith(...)` в обоих скриптах
держит модульный импорт безопасным — без него тест увёл бы смоук на живой прод.

собранный dist и деградирует в skip, а не в ложную зелень.

[Математик]: `permVersion` проверяется строгим неравенством `>` от снятого до
правки значения, а не «!= 0» — эпоха ловится корректно. HMAC-вектор фиксирован
детерминированно (тот же вход → тот же токен), плюс проверено, что смена секрета
меняет подпись при том же payload, а протухший TTL ядро отвергает. Квотирование
аргументов проверено на трёх злых случаях: пробел, апостроф, `$(whoami)`.

[Музыкант]: Не применимо — audio path не затронут. Единственное касание аудио
здесь — `dist/compare-audio` как статика: и это как раз главный содержательный
фикс, бандл из 120 wav вне git больше не сносится выкладкой.

[Верстальщик]: Не применимо — UI не затронут. Косвенно чинит выдачу статики:
