## Вердикт M0 (единственный — ПОРЯДОК)

```
корень  3  guardrails: hard (scoped-профиль + изоляция worktree + ветка
           origin/main) ПЕРВИЧНЫ; «мёрж утром» — вторичное следствие (research Q3:
           процесс без hard = театр)
           │
           ├─ 4  night:open советует ветку origin/main (парал. режим)   ┐ тулинг,
           └─ 5  night:close авто-handoff из коммитов (follow-up: теги)  ┘ без заседания,
                                                                          мерж после M1
параллельно, ортогонально цепочке:
        1  edit-class taxonomy (код→юнит / конфиг→линтер / инфра→plan / интегр→sandbox)
        2  гранулярность кванта + каденс чекпоинта (45–60 мин / логич. блок)
```

**Рёбра:** `3→4`, `3→5` (реализация нормы не опережает её определение).
**Порядок созыва:** M1 = 3 (фундамент) → тулинг 4+5 (парный инкремент, мерж после
M1) → M2 = 2 (каденс спорен) → M3 = 1 (словарь edit-class).
**Без консилиума:** 4 и 5 (тулинг, детерминированы после M1).
**Ограда:** узлы scripts-boundary (версионирование схемы, ретрай, реестр,
журнал прогонов) НЕ входят — другой контур.
