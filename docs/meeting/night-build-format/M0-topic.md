# Заседание `night-build-format` — M0 (установочное)

Задание: `docs/meeting/night-build-format/MEETING_BRIEF.md`.
Вход: `docs/tasks/research/night-build-format-v2.md` (проверен по коду).

M0 держит пять кандидатов в одной комнате и выносит **единственный** вердикт —
порядок зависимостей (DAG): что фундамент, что зависит, что параллельно, какие
кандидаты не требуют консилиума (тривиальный фикс/готовый канон), порядок M1..Mn.

## Единственный вопрос заседания M0

**O1 — Каков порядок зависимостей (DAG) между пятью доработками формата
night-build: какая правка фундамент, что от чего зависит, что параллельно, какие
исполнить без отдельного заседания, и в каком порядке созывать M1..Mn?**

## Кандидаты (простым текстом, ранг назначает M0)

1. Edit-class taxonomy: тест-гейт по классу правки (код → юнит; конфиг → линтер;
   инфра → plan; интеграция → sandbox). Регламентная правка.
2. Гранулярность кванта + каденс чекпоинта (риск/сложность/зависимость, 45–60 мин
   или логич. блок); различить вес NB-фаз. Регламентная правка.
3. Переформулировать guardrails: hard (scoped-профиль + изоляция worktree + ветка
   origin/main) первичны, «мёрж утром» вторичен. Регламентная правка (research Q3:
   процесс без hard = театр).
4. `night:open`: в параллельном режиме советовать ветку от origin/main. Тулинг,
   тривиальный фикс (эпизод 17.07: локальный main залочен).
5. `night:close`: генерировать merge-handoff из коммитов ветки (follow-up: теги).
   Тулинг.

## ⛔ ОГРАДА (run1 дрейфанул — 18.07)

Кандидатов РОВНО ПЯТЬ, перечислены выше. НЕ импортировать кандидаты соседнего
заседания `scripts-boundary` (17.07): версионирование СХЕМЫ артефактов, ретрай
транзиентных сбоев, реестр процессов, гейт свежести, журнал прогонов — это ДРУГОЙ
контур (S/F/J/R/Reg), он к формату night-build отношения не имеет. Если роль
называет узел не из пяти выше — это дрейф, вернуть к списку.

## Что должен вернуть M0

- DAG: рёбра «X — фундамент для Y».
- Корень (фундамент) и что параллельно.
- Кандидаты без консилиума (тривиальный фикс / готовый канон) — исполнять, не
  заседать по ним (кандидат 4 — вероятно сюда).
- Порядок созыва M1..Mn.
- Вердикт называет полный список посылок.
