## Вердикты (Р1–Р5, по ратифицированной цепи)

| Р | Фаза | Вердикт | Протокол |
|---|------|---------|----------|
| Р1 | Дом слоя | `docs/procedures/<id>/`: `README.md` + `MANIFEST.json` (`id`, `leadPersona`, `kitVersion`, `engines[]`, `precedents[]`); без кода и тестов; подвид `manifest-only` паттерна `GROUP_CONTAINERIZATION`; определения ≠ прогоны (`storm/`, `meeting/` не мигрируют); зуб `validateProcedure(dir)` в CI; первый жилец `ritual-evening` | `m1-home-r2` |
| Р2 | Словарь категорий | Полуоткрытый: ядро 4 (команда, задача, ответственность, цель) закрыто, расширение — ADR + LGTM владельца; единый машиночитаемый источник, `VOCABULARY.md` генерится; статья: имя·определение·род·маркер `@cat:`·предикат; чистый `check()` со списком нарушений, «0» — явно | `m2-vocabulary-r2` |
| Р3 | Граница слоёв | Ранги `процедура=0 < кит=1 < скрипт=2`, ссылки только вниз; легальность кодирует единый файл правил (один SHA); `checkLayerDirection` — блокирующий CI-зуб с отличимой ошибкой инструмента; кит-манифест — обязательный вход ревью; парсер ответственности: определение `docs/procedures/attribution/`, правила — файл правил, код — первый кит (pinned), маршрут: вердикт(`rulesSha`) → git-история → прецеденты | `m3-boundary-manual` |
| Р4 | Грамматика веток | `[<персона>/]<kind>/<slug>` + ортогональные теги формата; `kind` — закрытый словарь в файле правил (дубль `feature` умирает); каскад `resolveHolder(explicit, branchPersona) → persona\|CONFLICT\|MISSING` (`explicit` = слово ∨ `leadPersona` карточки при slug=id); красные исходы — возврат на доработку с подсказкой (T8); `night` — тег заморозки вне каскада; агент-префиксы вне канона (`Co-Authored-By`); история не переименовывается | `m4-grammar-manual` |
| Р5 | Миграция процедур | По касанию + первый залп из двух доноров: `ritual-evening` (Ангелина), `meeting` (Vesnin); `migrated = container ∧ vocabulary ∧ grammar` — производный предикат с провенансом `<persona>@<hash>`; реестр `docs/procedures/registry.json` (отдельный от задач), проекция генерится; немигрированные — `legacy`; держатель = `leadPersona` манифеста | `m5-migration-manual` |

## Порядок исполнения

Р1 → Р2 → Р3 → Р4 → Р5 той же цепью: дом до словаря, словарь до границы,
граница до грамматики, грамматика до залпа миграции.
