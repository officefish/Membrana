## Решено владельцем — переспорить нельзя (тезисы шторма)

- **Шипование — только через скилл**; ответственный: явное слово → иначе из ветки (T1, T2).
- **Ревью ведёт один из пяти, назначенный главным**: антипаттерны → пропуск/блок;
  работает его собственная память (T3, T4, T5).
- **Процедурный контент → ведущая персона процедуры** (шторм → Ангелина); по сложным
  коммитам ответственность распределяет парсер на code-review (T6).
- **`night` — тег заморозки**: ничего в репозиторий до флага «ночь закончилась» (T7).
- **Ветка вне конвенции → возврат агенту на доработку** (T8).
- **Скрипты ≠ процедуры**: скрипты — функции в плоском контейнере; процедуры —
  спроектированные сущности над категориями (T10, T11).
- **Процедуры управляют скриптами через ссылку; группировка — виртуальные наборы, киты**;
  контейнеру процедуры не нужны собственные скрипты и тесты (T11, T12).
- **Ритуалы — процедуры** (T13).
- **Киты версионируются с момента появления** (паттерн PINNED_SUBGRAPH_VERSIONING);
  контейнеры — нет.
- **Легаси-база в целом** (перенос всего существующего на новые рельсы) — вынесена
  за пределы этого контура отдельным решением владельца.
