## Порядок (вердикт M0 — **РАТИФИЦИРОВАН владельцем 2026-07-21**)

Линейная цепь, независимых пар нет, топологическая сортировка единственна:

```
1 дом слоя → 2 словарь категорий → 3 граница слоёв/киты → 4 грамматика веток → 5 миграция процедур
```

Протокол M0: [`m0-order`](../../seanses/procedural-layer-m0-order-2026-07-21.md).
