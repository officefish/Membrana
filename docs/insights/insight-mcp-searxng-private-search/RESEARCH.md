# Research: SearXNG — приватный keyless веб-поиск для агентов

> Источник: perplexity-api

## Q1 — Landscape

**Запрос:** SearXNG — приватный keyless веб-поиск для агентов: industry landscape, open-source approaches 2024-2026

**Выжимка:**

SearXNG — это бесплатный метапоисковик с открытым исходным кодом (AGPL-3.0), который агрегирует результаты до 276 поисковых систем без использования собственного индекса, метаданных пользователей или создания пользовательских профилей[2][7][8].

### Industry Landscape (2024–2026)
В 2024–2026 годах SearXNG стал ключевым решением для **keyless-веб-поиска для AI-агентов**, позволяя избежать дорогостоящих API-взносов (например, $10 за 1000 запросов у Anthropic) и ограничений по частоте (rate limits)[3].
- **Экономическая эффективность**: Интеграция через MCP (Model Context Protocol) с SearXNG убирает плату $15–30/мес за поиск и устраняет ошибки `API Error: Rate limit reached`[3].
- **Прозрачность и качество**: Агенты получают 23 результата из 5 движков (вместо одного источника у провайдеров облачного поиска) и полную видимость времени ответа через `Server-Timing`[3].
- **Приватность**: SearXNG не собирает метаданные, не отслеживает запросы и рандомизирует `User-Agent` и `cookies`, делая каждый запрос похожим на новый вход с нового компьютера для Google[1][3].

### Open-Source Approaches (2024–2026)
SearXNG реализует передовые подходы обеспечения анонимности и совместимости в экосистеме open-source:
| Подход | Описание и технологии |
|--------|----------------------|
| **Агрегация систем** | Объединяет результаты от 70–276 поисковых сервисов (Google, Bing, DuckDuckGo, Brave, ~245 других) в единый список[2][6][7] |
| **Keyless-поиск** | Не требует API-ключей: работает как парсер (мост) между frontend-ом пользователя и API поисковиков, отрезая встроенные трекеры и рекламу[1][3] |
| **Анонимизация** | Поддержка маршрутизации через **Tor** (`outgoing.using_tor_proxy: true`) и рандомизация TLS-шифров (`shuffle_ciphers()`) для защиты от фингерпринтинга[3] |
| **Категоризация** | Категории поиска: стандартные («Веб», «Изображения», «Новости») и специализированные («Социальные сети», «Музыка», «IT», «Файлы»)[2][6] |
| **Доступность** | Установка на собственный сервер или ПК через Docker, с публичными экземплярами на `searx.space` и возможностью работы в локальной сети[2][6] |

### Ограничения и особенности
- SearXNG **не имеет прямой интеграции** с системами домашней автоматизации и не является частью экосистем IoT[2].
- Поиск работает через цепочку `localhost → ваш IP → поисковики`, и поисковики видят ваш IP, но с пустыми `cookies` и заголовком `DNT` (Do Not Track)[3].
- Для настройки требуется создание секретного ключа (например, через `openssl rand -hex 32`) и настройка файла `settings.yml`[1][6].

SearXNG остается лидером в open-source метапоиске, обеспечивая прозрачность, отсутствие rate limits и бесплатность для AI-агентов в 2024–2026 годах[3].

## Q2 — Fit (Membrana)

**Запрос:** SearXNG — приватный keyless веб-поиск для агентов: fit with Web Audio, edge recording, zero-shot audio, TypeScript monorepo

**Выжимка:**

SearXNG — это бесплатный метапоисковик с открытым исходным кодом, который агрегирует результаты от множества поисковых систем (до 276), не использует метаданные пользователей и не отслеживает поисковые запросы, что обеспечивает приватность и отсутствие профилирования [2][9].

У проекта на GitHub более 25 500 звёзд, и он работает как «мост» между собственным frontend-ом и frontend-ом поисковиков (или их API), отправляя запросы в Google, DuckDuckGo, Brave и другие системы, собирая результаты и возвращая их пользователю без рекламы и встроенных трекеров [1][3].

SearXNG не имеет собственного индекса, полностью настраиваем, позволяет выбирать нужные поисковые системы, поддерживает маршрутизацию через Tor и прокси для усиленной анонимности, а также предлагает категоризированный поиск (веб, изображения, видео, новости, музыка, IT и др.) [2][3][5].

Он можно установить на свой сервер или персональный компьютер с помощью Docker (`docker-compose up -d`), получить доступ через локальный адрес (`http://localhost:8080`) или через HTTPS с SSL-сертификатами (например, через Let’s Encrypt и Nginx) [6][7].

Для приватного доступа в масштабируемых сетях можно использовать Tailscale с Magic DNS и TLS-сертификатами, а также интегрировать SearXNG в Chrome как поисковую систему с ключевым словом (например, `sx`) и URL вида `https://search.<ваш-tailnet>.ts.net/search?q=%s` [6].

SearXNG поддерживает более 200 поисковых систем (по умолчанию включено около 85), удаляет приватные данные из запросов, отправляемых поисковым сервисам, и не создаёт пользовательские профили, что делает его идеальным решением для тех, кто ценит конфиденциальность [2][5].

Проект также позволяет использовать каталогированные категории поиска, включая нестандартные, такие как «Социальные сети», «Файлы», «Наука», и предоставляет расширенные параметры фильтрации по диапазону дат и типу контента [2][5].

 В отличие от платных сервисов веб-поиска (например, $10 за запрос в некоторых AI-инструментах), SearXNG обеспечивает бесплатный веб-поиск для агентов и AI-систем, экономя ресурсы и сохраняя анонимность [3].

 Для максимальной безопасности SearXNG можно запускать локально, генерируя секретный ключ для развёртывания службы Search, и использовать его в конфигурационных файлах (например, в `config.yml` или `docker-compose.yml`) [4].

 SearXNG активно используется в проектах вроде OpenWebUI, хотя в некоторых случаях может наблюдаться проблема с повторным обращением к одним и тем же сайтам, если не настроена корректная интеграция [10].

 В целом, SearXNG — это надёжный, бесплатный и приватный инструмент для веб-поиска, который можно интегрировать в различные системы, включая AI-агентов, с минимальными усилиями и без затрат на лицензирование [2][3][9].

## Q3 — Risk

**Запрос:** SearXNG — приватный keyless веб-поиск для агентов: risks latency cost privacy licensing flakiness team velocity

**Выжимка:**

SearXNG — это бесплатный метапоисковик с открытым исходным кодом (лицензия), который не имеет собственного индекса, фокусируется на приватности (не отслеживает пользователей и не собирает данные), но требует технических знаний для самостоятельного развертывания и может быть идентифицирован базовыми поисковыми системами при одиночном использовании[1][2][3].

Ниже приведен анализ ключевых аспектов и рисков, указанных в вашем запросе:

| Аспект | Состояние и риски | Описание и источники |
| :--- | :--- | :--- |
| **Приватность (Privacy)** | **Высокая, но с нюансами** | SearXNG не собирает персональные данные, не создает профили и удаляет приватные данные из запросов[1][4]. Однако, если вы единственный пользователь своего экземпляра, поисковые запросы могут быть легче идентифицированы базовыми поисковыми системами (например, Google), так как они видят ваш IP[1]. |
| **Латентность (Latency)** | **Зависит от инфраструктуры** | Поиск идет через `localhost` к вашему IP, затем к поисковикам, минуя посредников (как Anthropic), что дает прозрачность timing через `Server-Timing`[2]. Латентность определяется скоростью ответа конкретных движков (Google, Bing и др.) и задержками сети, так как SearXNG не имеет собственного быстрого индекса[2]. |
| **Стоимость (Cost)** | **$0 (безлимитно)** | Это полностью бесплатный инструмент, который устраняет плату за поиск ($0.01 за запрос в WebSearch) и лимиты запросов (rate limits), что экономит ~$15–30/мес при_active использовании[2]. |
| **Лицензирование (Licensing)** | **Open Source (Бесплатно)** | Проект с открытым исходным кодом, который можно установить на свой сервер или ПК[1][2]. |
| **Флакишнесс (Flakiness)** | **Риск нестабильности** | Поскольку SearXNG является «парсером» (мостом) между фронтендами поисковиков, а не официальным API, он может быть нестабильным при изменении верстки сайтов или блокировок поисковиками[3]. Также сам SearXNG пока не поддерживает аутентификацию, что позволяет другим обнаруживать и злоупотреблять экземпляром при публичном доступе[6]. |
| **Техническая скорость (Team Velocity)** | **Низкая для пользователя** | Требует ~10 минут настройки Docker и Nginx, но обслуживание требует определенных технических навыков (self-hosting)[2][6]. Вектор развития проекта не связан с экосистемами домашней автоматизации и ориентирован на веб-поиск[1]. |

**Ключевые рекомендации:**
*   **Анонимность:** Для максимальной защиты используйте маршрутизацию через прокси или Tor (`outgoing.using_tor_proxy: true`) и подмену TLS-шифров (`shuffle_ciphers()`), чтобы избежать TLS-фингерпринтинга[2][4].
*   **Безопасность публичного доступа:** Если разворачиваете сервис в публичной сети, **обязательно** настройте HTTP Basic Authentication через Nginx, так как штатная аутентификация в SearXNG отсутствует[6].
*   **Ограничения API:** Настройка `limiter: false` может быть необходима для приватных экземпляров, так как лимитирование в SearXNG мешает доступу к API[6].

SearXNG отлично подходит для агентов, требующих бесплатного и приватного поиска, но требует мониторинга стабильности парсинга и правильного настройки безопасности при публичном доступе[2][3].
