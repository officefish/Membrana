[Математик]: Ключевой численный факт корректен и это сердцевина вердикта: луч 2.5° при σ(азимут) в пару градусов на дистанции 1 км промахивается — геометрия неумолима. Порог возврата σ ≤ 0.6° выбран разумно (запас ~4× относительно ширины луча по угловому бюджету). Но подчеркну: **моно-микрофон физически не даёт DOA** — азимут требует разнесённой апертуры (микрофонная решётка) и это отдельная крупная задача с собственным шумовым и калибровочным бюджетом. То есть «σ ≤ 0.6°» — не «допилить фильтр», а новый эпик пеленгации со своей мат-моделью (корреляционный TDOA / MUSIC-класс методы, оценка Cramér–Rao для достижимой точности). Пока эшелон 0 (детекция P≥85/R≥90) не закрыт — считать угловую точность преждевременно. Мат-содержания для внедрения сейчас нет.
