# сенсорный узел

Микрофон + плата уровня Raspberry Pi + точное время.

**Терминология.** Говорим: **сенсорный узел** / **узел**. Не говорим: «нода», «датчик», «устройство», «сенсор» (в одиночку), «микрофон» (как весь узел).

Источник: живое чтение `docs/comms/canon/GLOSSARY.md`. Факты и цифры — из `FACTS_SHEET.md`;
неподтверждённые метрики на внешний материал не выносятся.
