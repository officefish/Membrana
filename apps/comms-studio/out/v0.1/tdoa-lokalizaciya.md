# локализация по разнице времён прихода

Вычисление координат по разнице времён прихода звука на узлы.

**Терминология.** Говорим: **локализация по разнице времён прихода (TDOA)**. Не говорим: «триангуляция», «радар».

Источник: живое чтение `docs/comms/canon/GLOSSARY.md`. Факты и цифры — из `FACTS_SHEET.md`;
неподтверждённые метрики на внешний материал не выносятся.
