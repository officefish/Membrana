# Documentation workflow

## Слои

| Слой | Tool | Роль |
|------|------|------|
| Publish | **Mintlify** (`apps/docs`) | Node reference, concepts, cookbooks |
| Spec | **ChatPRD** MCP | PRD, DoD, alignment |
| Corporate | **Atlan** MCP | Glossary, корп. термины, metadata search |
| Agent truth | `docs/catalog/` | Обязательно перед правками кода |
| Visual | **Cursor Docs Canvas** | Architecture / streaming diagrams |

## Локальная разработка docs

```bash
yarn docs:dev
# http://localhost:3333
```

## MCP Phase D

```bash
yarn mcp:phase-d              # отчёт + generated JSON
yarn mcp:phase-d:install      # merge tier4 в ~/.cursor/mcp.json
```

Tier4 servers (OAuth при первом use):

| Server | URL |
|--------|-----|
| mintlify-reference | `https://mintlify.com/docs/mcp` |
| mintlify-admin | `https://mcp.mintlify.com` |
| chatprd | `https://app.chatprd.ai/mcp` |
| atlan-docs | `https://docs.atlan.com/mcp` |
| atlan | `https://mcp.atlan.com/mcp` |
