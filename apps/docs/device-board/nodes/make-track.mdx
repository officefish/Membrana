# MakeTrack

## Pins

| in | `recorder` | RecorderRef |
| in | `samples` | AudioSampleRef[] (batch) |
| in | `slice` | RecordingSliceRef (v0.7) |
| out | `track` | TrackRef |

## Runtime

На exec: host upload/encode track → `TrackRef` в runtime store. Terminal node (DBC4).
