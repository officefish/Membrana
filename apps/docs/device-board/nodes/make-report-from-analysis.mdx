# MakeReportFromAnalysis

Отчёт по **одному** детектору любой модальности. Вход обобщён (ADR-0006): принимает
`FftTrendAnalysisRef` (спектр) ИЛИ `EnsembleAnalysisRef` (нейро) через санкционированный
union `DetectionAnalysisRef`. Одиночная модальность не носит `MakeCombinedReport` —
узел честен по имени.

## Pins

Exec + `reporter` + `analysis` (`DetectionAnalysisRef`) in → `report` out.

## Runtime

- Спектральный вход (`FftTrendAnalysisRef`) → схема отчёта `trends-fft/v0.1`.
- Нейро-вход (`EnsembleAnalysisRef`) → схема `neuro-detection/v1` (host-хук
  `makeReportFromEnsembleAnalysis`, детекция резолвится исполнителем из ensemble-store).

Обратная совместимость: union шире прежнего `FftTrendAnalysisRef` — сохранённые
графы остаются валидны.
