# Мастерская бестиария — обзор

**Монитор (это приложение)** зеркалит дом в git. Дом **не** переезжает сюда.

| Слой | Смысл | Где истина (git) |
|------|--------|------------------|
| **Класс** | `defectClass` линзы | `docs/audit/bestiary/registry/BESTIARY_LIST.md` · engines `scripts/lib/lens-bestiary.mjs` |
| **Шаблон** | абстракт антипаттерна | `docs/audit/bestiary/antipatterns/<id>.md` |
| **Ловушка** | prompts + pure scripts | `docs/audit/bestiary/traps/` · индекс `registry/TRAPS_LIST.md` |
| **Улов** | пойманная гранула (≠ specimen) | `docs/audit/bestiary/registry/CATCH_LIST.md` |
| **Specimen** | фикстура forcing function | `docs/audit/bestiary/specimens/<class>/` |

Цепочка: шаблон → ловушка → улов. Specimen — отдельно (не гранула улова).

## Лемма: git = истина · Mintlify = монитор

- Канон и правки — только в `docs/audit/bestiary/` (и engines снаружи дома).
- Эти страницы **могут отставать**; сверка — по git-путям (и SHA при необходимости).
- **Глубина W3:** thin mirror — overview + зеркала примеров улова/ловушки.

  расширение каталога классов / cookbooks — follow-up.
