# AI Стратегия Membrana

## Уровни автоматизации

### L1: Подсказки (Cursor)

- Автокомплит кода
- Объяснение контекста пакетов
- Быстрые фиксы импортов

### L2: Авто-исправление (Pre-commit)

- Форматирование
- Линтинг
- Базовые тесты
- Фикс импортов между пакетами

### L3: CI Ревью (GitHub Actions)

- Критика кода
- Проверка совместимости пакетов
- Предсказание конфликтов

### L4: Авто-документация

- Генерация API docs
- Обновление README пакетов
- Создание примеров

### L5: Авто-тестирование

- Генерация unit-тестов
- Нахождение edge случаев
- Создание интеграционных тестов между пакетами

### L6: Самоисцеление (Production)

- Анализ ошибок в рантайме
- Автоматический откат
- Создание issue с контекстом решения

## Метрики эффективности AI

- Время от коммита до PR review: цель < 2 мин
- Процент сгенерированных тестов: цель > 60%
- Авто-фикс ошибок в CI: цель > 40%
- Время инцидента до авто-отката: цель < 30 сек
